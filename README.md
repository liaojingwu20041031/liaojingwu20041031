## About me

我是来自三峡大学计算机科学与技术专业的开发者，关注 **机器人系统、具身智能与 AI Agent 工程**。

我喜欢把跨层技术真正接通：从 **STM32 / CAN / 传感器驱动**，到 **ROS 2 / SLAM / Nav2 / 视觉感知**，再到 **多模态大模型、Agent 工具链、移动端与现场 UI**。目标不是停留在 Demo，而是做出能够联调、部署、比赛和持续迭代的完整系统。

> Turning ideas into running systems — from embedded control to intelligent agents.
